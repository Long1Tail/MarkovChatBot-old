# Что это?

Это небольшой telegram-бот, предназначенный для генерации сообщений по команде. В его основе - цепь Маркова. Он способен обучаться на словах, увиденных в личных сообщениях.

# Как это запустить?

Всё очень просто.

## 1.  Убедись, что установлен Python

Если что, скачать его можно вот [здесь](https://www.python.org/downloads/release/python-31210/)

## 2. Создай и активируй виртуальное окружение

Это делается следующими коммандами:

```
python -m venv .venv
./.venv/Scripts/activate
```

## 3. Установи зависимости

Введи в терминал команду 

```
pip install -r requirements.txt
```

## 4. Создай файл с токеном твоего бота

В папке с проектом создай файл `config.py`

В него помести всего 1 строку:
```
TOKEN = <Замени этот текст на токен, полученный от @BotFather>
```

Как его получить, можешь почитать вот [здесь](http://botcreators.ru/blog/botfather-instrukciya/)

## 5. Запусти бота

Это тоже делается одной командой

```
python main.py
```
